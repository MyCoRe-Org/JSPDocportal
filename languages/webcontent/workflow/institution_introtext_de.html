Allgemeine Hinweise:
Wenn anstelle einer Person eine externe Einrichtung (z.B.  AnInstitute) Dokumente in die Digitale Bibliothek einstellen möchte, kann auch für diese ein Datensatz angelegt werden.
Anhand dieses Datensatzes können die Dokumente eindeutig der Einrichtung zugeordnet werden.
Es wird empfohlen, für alle universitären Einrichtungen, die Dokumente veröffentlichen wollen, entsprechende Datensätze anzulegen.

Vorgehensweise:
Starten Sie zunächst den Workflow über den Link „Neue Institution anlegen“. Dann wird ihrer Arbeitsmappe ein neuer Eintrag hinzugefügt.
Anschließend können Sie den Datensatz bearbeiten.

Nach dem Editieren haben Sie die Möglichkeit, die Zugriffsrechte für den Datensatz zu bearbeiten und sich eine Vorschau für ihre Daten anzeigen zu lassen.

Sind die Daten vollständig, klicken Sie den Link „Daten der Institution an die Digitale Bibliothek senden“. Die Daten werden an die Bibliothek gesendet. Dort prüft ein Bibliothekar auf Vollständigkeit und Korrektheit und gibt den Datensatz frei. Darüber werden Sie per Email informiert.

Sie können jedoch sofort damit beginnen, für diese Institution Dokumente einzustellen.
