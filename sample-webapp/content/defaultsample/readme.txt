
Default sample data for the MyCoReSample
========================================

This collection contains some simple sample datas for the MyCoReSample
installation. With simple commands you can install them to your
installed MyCoReSample.

Installation:

1. ant info --> check the output that all varibles ar setting

2. ant load --> load all of this sample data to your MyCoReSample


Deinstallation:

1. ant info --> check the output that all varibles ar setting

2. ant remove --> remove all of this sample data from your MyCoReSample


Remember: Alles wird gut!
