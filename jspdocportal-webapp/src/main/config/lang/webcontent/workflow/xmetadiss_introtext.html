Allgemeine Hinweise:
Die Universitätsbibliothek bietet Doktorandinnen und Doktoranden die Möglichkeit, ihre Dissertation im Internet zu veröffentlichen.
Als zusätzliche Abgabeform wird neben den herkömmlichen Abgabeformen auch die elektronische Veröffentlichung von Dissertationen in allen Fachbereichen als Abgabe akzeptiert.

Vorgehensweise:
Um Ihre Dissertation elektronisch zu publizieren, benötigen Sie einen eigenen Nutzeraccount. Anschließend könne Sie über den Menüpunkt „Dissertation anmelden“ den Workflow starten.

Es wird eine URN (eine Art ISBN für elektronische Dokumente) für Sie reserviert. Diese sollten sie idealer Weise in den Text ihrer Dissertation aufnehmen. Mittels der URN kann bei der Deutschen Bibliothek jederzeit der aktuelle Standort der Datei ermittelt werden.

Anschließend können Sie die Metadaten für ihre Dissertation eingeben. Bis zur Abgabe können Sie die Daten beliebig oft editieren. Während der Bearbeitungsphase haben nur Sie Zugriff auf das Dokument. Es ist nur durch Sie einsehbar.

Zusätzlich zu den Metadaten müssen Sie den Volltext der Dissertation auf unseren Server laden.  Er sollte möglichst als PDF-Datei vorliegen. Weitere Dateien, die zu der Dissertation gehören, sollten Sie als Zip-Datei packen und als Anhang hochladen.

Wenn alle Angaben vollständig sind und Sie die Dateien auf den Server geladen haben, können Sie ihre Dissertation an die Digitale Bibliothek senden. Erzeugen Sie bitte auch das Abgabeformular, dass sie in der Hochschulschriftenstelle vorlegen.

Nach dem Absenden an die Digitale Bibliothek können Sie an den Metadaten keine Änderungen mehr vornehmen. Ihre Daten werden durch einen Bibliothekar auf Vollständigkeit und Korrektheit überprüft. Anschließend gibt er die Dissertation frei und sie kann durch andere Nutzer der Digitalen Bibliothek gelesen werden. Die Daten werden der Deutschen Nationalbibliothek übermittelt.

Sollten Probleme auftreten, werden wir per Mail mit Ihnen Kontakt aufnehmen.
