Allgemeine Hinweise:
Selbständige und unselbständige Publikationen werden bei der Erfassung der Metadaten unterschieden.
So ist für einen Zeitschriftenartikel, als unselbständige Publikation beispielsweise die Zeitschrift anzugeben, in der dieser Artikel erschienen ist.

Vorgehensweise:
Um Ihre Publikation elektronisch zu veröffentlichen, benötigen Sie einen eigenen Nutzeraccount.
Außerdem sollten Sie, wenn für sie noch kein Autorendatensatz existiert, einen solchen anlegen.

Anschließend können Sie über den Menüpunkt „Publikation anmelden“ den Workflow starten.
Danach wir der Typ der Publikation abgefragt.

Es wird eine URN (eine Art ISBN für elektronische Dokumente) für Sie reserviert. Diese sollten sie idealer Weise in den Text ihrer Publikation aufnehmen. Mittels der URN kann bei der Deutschen Bibliothek jederzeit der aktuelle Standort der Datei ermittelt werden.

Anschließend können Sie die Metadaten für ihre Publikation eingeben. Bis zur Abgabe können Sie die Daten beliebig oft editieren. Während der Bearbeitungsphase haben nur Sie Zugriff auf das Dokument. Es ist nur durch Sie einsehbar. Jedem Eingabefeld wurde ein Hilfebutton (?) zugeordnet, der einen kurze Beschreibung über den Inhalt des Feldes anzeigt.

Zusätzlich zu den Metadaten müssen Sie den Volltext der Publikation auf unseren Server laden.  Er sollte möglichst als PDF-Datei vorliegen. Natürlich können auch weitere Dateien hochgeladen werden.

Außerdem können sie den Zugriff auf das Dokument einschränken. So können Sie beispielsweise festlegen, das nur registrierte Benutzer oder nur Nutzer aus dem Hochschulnetz Zugriff auf das Dokument haben sollen. Dieses Zugriffsrecht bezieht sich nur auf den Volltext (die Dateien). Metadaten können durch jeden Nutzer der Digitalen Bibliothek eingesehen werden.

Wenn alle Angaben vollständig sind und Sie die Dateien auf den Server geladen haben, können Sie ihre Publikation an die Digitale Bibliothek senden. Nach dem Absenden an die Digitale Bibliothek können Sie an den Metadaten keine Änderungen mehr vornehmen. Ihre Daten werden durch einen Bibliothekar auf Vollständigkeit und Korrektheit überprüft. Anschließend gibt er die Publikation frei und sie kann durch andere Nutzer der Digitalen Bibliothek gelesen werden.

Sollten Probleme auftreten, werden wir per Mail mit Ihnen Kontakt aufnehmen.

Technische Hinweise:
Für die Veröffentlichung auf dem Server sollte die Publikation als PDF-Datei vorliegen. Der Zugriff auf die Datei sollte nicht beschränkt und sie sollten alle verwendeten Fonts (Schriften) und Graphiken enthalten. Weitere Dateien können Sie in eine Zip-Datei packen und als Anhang ebenfalls auf den Server laden.

Rechtliche Rahmenbedingungen:
Mit der Veröffentlichung in der Digitalen Bibliothek verbleiben die Urheberrechte bei Ihnen. Die Einhaltung von Urheber- und Verwertungsrechten Dritter liegt in der Verantwortung der Autoren bzw. Herausgeber der elektronischen Dokumente.
